# Generador de Cotizaciones VAUTEC

Sistema profesional de generación de cotizaciones con base de datos local y generación de documentos en PDF.

## Características

- Gestión de clientes
- Múltiples conceptos por cotización
- Tabla de alcances del proyecto
- Generación de PDF
- Exportación en paquete ZIP
- Base de datos en localStorage del navegador

## Tecnología

- React 18
- Tailwind CSS
- JSZip para archivos comprimidos
- Single Page Application (SPA)

## Uso

Accede a la aplicación y comienza a crear cotizaciones profesionales.
